网站更新发布到github的命令

第一步：在Finder中找到网站目录，单击选中文件夹，cmd+c。打开mac电脑终端，输入cd，然后cmd+v。

第二步：在终端输入如下指令

第一种方法

echo "# gt8999.github.io" >> README.md
git init
git add .
git commit -m "first commit"
git remote add origin https://github.com/gt8999/gt8999.github.io.git
git push -u origin master

绑定域名指令：

echo 'www.gtilili.com' >> CNAME
git add --ignore-removal .
git commit -m "cname"
git push -u origin master

更新文章后推送指令：

git add --ignore-removal .
git commit -m "cname"
git push -u origin master
