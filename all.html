工业机器人实训室注意事项

机器人操作

  1. 在编程、调试、检修时，必须将机器人置于手动模式，且手动模式下机器人速度限制为最大250mm/s，在安全保护空间之内工作时，应始终以手动速度进行操作，禁止使用全速手动模式(100%)！
  2. 注意机器人的旋转轴，机器人运动时应与各轴保持距离，防止头发或衣服被缠扰，在使用使动装置移动机器人或在机器人工作范围内进行其他工作时，存在机器人意外移动的风险。
  3. 在机器人的运动编程时，第一次试运行之前应检查是否存在潜在的碰撞风险。
  4. 机器人处于自动模式时，按下启动按钮之前应确保机器人工作范围内无人员活动。
  5. 机器人运行中发生任何意外或运行不正常时，立即使用E-Stop键（急停按钮），使机器人停止运行。
  6. 注意工件和机器人系统的高温表面，机器人电机长期运转后温度很高。
  7. 调试机器人时应确保夹具夹好工件，谨防夹具打开，工件脱落导致人员伤害或设备损伤，机器人停止运动时，手臂上不能夹取工件或任何物品。
  8. 进入机器人工作空间时，必须始终随身携带示教器，以防其他人此时操作机器人造成人身伤害。
  9. 突然停电后，要手动及时关闭机器人的主电源和气源。
  10. 实验完毕，要按规定程序关闭机器人，清点和整理器材，打扫环境，经许可方可离开实验室。

机器人示教器操作

1.小心操作，不要摔打、抛掷或重击示教器，这样会导致其破损或故障。
2.如果示教器受到撞击，应验证其安全功能（使动装置和紧急停止）是否仍然有效。
3.实验结束后，应将其置放于立式壁架上存放，防止意外掉落。
4.使用和放置示教器时都应避免其电缆线将人绊倒。
5.禁止使用锋利的物体点击触摸屏，这样会使触摸屏受损，应使用手指或触摸笔操作。
6.应定期清洁触摸屏，灰尘和小颗粒可能会挡住触摸屏造成故障。
7.禁止使用溶剂、洗涤剂或擦洗海绵清洁示教器。
8.未连接USB设备时应盖上USB端口的保护盖，端口长时间暴露到灰尘中，会中断或发生故障。
9.在不移动机器人或不运行程序时，须及时释放使能键。

现代电气控制技术分中心视频宣传解说词

现代电气控制技术分中心是河北唐山职业技能公共实训中心的六大中心之一。分布于学院实验楼A座、4号实训楼和信息楼七层，使用面积约7223平方米，规划有16种类别共计34个实训教室，配有先进的实训设备457台套，具有高技能人才培养工位590个，能够满足6个职业工种的鉴定要求，还可承接多个专业的省级，国家级职业技能竞赛。

电气分中心培训内容涵盖现代电气控制领域中的PLC控制、人机界面、机电一体化、工业以太网、工业机器人、机器视觉等高新技术，同时还包括电力拖动控制、电气装置、电子产品设计装调、机床线路维修、液压与气动、工厂供配电、电梯维修等职业工种技能的培训。

中心实训室按照多功能区域划分，硬件上具有设备先进、种类全面、培训工位充足的特点，软件上采用模块化培训课程体系、开发了配套教材、信息化教学资源，依托电气系双师教师和骨干教师队伍，师资力量过硬。

海缸新入蓝吊一条

今日小海缸新入s号蓝吊一条，照了两张照片留作纪念

黄尾副刺尾鱼（学名：Paracanthurus hepatus）体侧扁，口小。鱼体呈鲜艳的宝蓝色，并有明显调色盘状黑带；背鳍与臀鳍皆为宝蓝镶黑色宽边；尾柄与尾鳍皆为鲜黄色，尾鳍上下叶具黑边，此黑边和鱼体黑色区域相连，使后方隔离形成三角形的黄色区域，尾柄棘亦在此区内。尾柄两侧各具一根硬棘，尾棘鞘不明显。头部上半部散布小黑点，有时不太明显。鱼体越大体色越淡。
该鱼主要分布于印太海域南中国海、印度尼西亚、马尔代夫、毛里求斯及南太平洋所罗门群岛等多地，且产量很大，为分布地区的习见鱼类。多活动于珊瑚礁区，觅食有机物碎屑等，会藏到礁石中躲避敌害。为一著名观赏鱼类。

网站更新发布到github的命令

第一步：在Finder中找到网站目录，单击选中文件夹，cmd+c。打开mac电脑终端，输入cd，然后cmd+v。

第二步：在终端输入如下指令

第一种方法

echo "# gt8999.github.io" >> README.md
git init
git add .
git commit -m "first commit"
git remote add origin https://github.com/gt8999/gt8999.github.io.git
git push -u origin master

绑定域名指令：

echo 'www.gtilili.com' >> CNAME
git add --ignore-removal .
git commit -m "cname"
git push -u origin master

更新文章后推送指令：

git add --ignore-removal .
git commit -m "cname"
git push -u origin master
